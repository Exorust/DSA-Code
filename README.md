## Getting Started

Each Week Folder contains the question list and their solutions.

### Prerequisites

Either gcc or clang.

### Running

Use this to compile:
```
$ gcc q-id.c -o name.out
```

Then use this to execute:
```
$ ./name.out
```

Look into the pdf to the corresponding question for its respective inputs and outputs.
